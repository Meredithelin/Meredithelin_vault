## 如何 Fork 这些项目

运行以下脚本，可将上述所有项目一键 Fork 到你的 GitHub 账号：

```bash
# 1. 获取 GitHub Token（需要 repo 权限）
# 访问：https://github.com/settings/tokens

# 2. 运行脚本
python3 copilot/scripts/fork_top_github_repos.py --token YOUR_GITHUB_TOKEN

# 或使用环境变量
export GITHUB_TOKEN=your_token_here
python3 copilot/scripts/fork_top_github_repos.py
```
